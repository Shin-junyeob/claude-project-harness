# 코딩 컨벤션 (공통)

모든 프로젝트에 공통 적용되는 코딩 규칙. 프로젝트별 예외는 같은 폴더에 별도 규칙으로 추가한다.

## 언어
- 기본 언어는 **Python 3** (별도 명시가 없는 한).
- 외부 의존성은 `venv/` 가상환경에 설치한다.

## 네이밍
- 파일·함수·변수: `snake_case`
- 클래스: `PascalCase`
- 상수: `UPPER_SNAKE_CASE`
- 의미가 드러나는 이름을 쓴다. 약어 남용 금지.

## 스타일
- 들여쓰기: 스페이스 4칸.
- 한 함수는 한 가지 일만 한다.
- 매직 넘버 대신 상수로 분리한다.
- 주석은 "무엇"이 아니라 "왜"를 설명한다. 과도한 주석 금지.

## 에러 처리
- 예외는 삼키지 말고 의미 있게 처리하거나 상위로 전달한다.
- 에러 발생 시 반드시 `logs/`에 기록한다. (→ [doc_update_rules.md](doc_update_rules.md))

## 구조
- 구현 코드는 반드시 `src/`에만 둔다. (→ [project_structure.md](project_structure.md))
