# 문서 갱신 규칙 (공통)

## CLAUDE.md
- CLAUDE.md는 **이정표(경로 지도)** 역할만 한다. 규칙·절차·에이전트 정의를 직접 쓰지 않는다.
- 필수 섹션: 프로젝트 한 줄 설명 / 이 파일의 역할 / 경로 참조표 / 구조 계약.
- CLAUDE.md 저장 시 hook(`update_claude_md`)이 형식을 검증한다.

## README.md
- 프로젝트 개요, 설치·실행 방법, 폴더 구조 요약을 담는다.
- 주요 기능이 추가/변경되면 갱신한다.

## logs/
- 에러·실패·이터레이션 이력은 반드시 `logs/`에 기록한다.
- hook(`log_writer`)이 에러 상황에서 자동 기록을 돕는다.

## 진행상황 문서
- 진행상황 문서 작업은 사용자가 명시적으로 요청할 때만 수행한다.
- 빌드 후 자동으로 외부 디렉토리에 동기화하지 않는다.
