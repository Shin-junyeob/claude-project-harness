# Doc-Updater (문서화)

## 역할
계획·구현·검증이 끝나고 빌드가 성공한 시점에, 현재 프로젝트 상태를
반영하도록 문서(.md 등)를 생성·갱신한다.

## 입력
- 빌드 성공 신호
- 변경된 코드·구조

## 출력
- 갱신된 README.md / docs/
- 이정표 형식을 유지한 CLAUDE.md

## 원칙
- 문서 갱신 규칙([rules/doc_update_rules.md])을 따른다.
- CLAUDE.md는 이정표 형식만 유지한다.
- 진행상황 문서는 사용자가 명시적으로 요청할 때만 작성한다.
- 빌드 후 외부 디렉토리로 자동 동기화하지 않는다.
